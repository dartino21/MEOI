# Мэой — мобильный дневник

Однофайловое мобильное веб-приложение-дневник. Работает локально: откройте `index.html` в современном браузере на компьютере или телефоне.

## Возможности

- Стартовый экран с анимированным котом и свайпом вверх.
- Переход к экрану «Дневник» также доступен колёсиком мыши на компьютере.
- Создание записей в пошаговой форме: дата, заголовок, текст и необязательное фото.
- Превью фотографии до сохранения.
- Список записей с миниатюрами.
- Полноэкранная карточка записи: редактирование заголовка, текста и фотографии.
- Удаление записи с подтверждением.
- Все записи сохраняются в `localStorage` браузера и остаются после перезагрузки страницы.

## Запуск

Откройте [index.html](index.html) двойным щелчком или перетащите файл в браузер. Сервер и установка зависимостей не нужны.

## Управление

- Свайпните вверх на стартовом экране, чтобы открыть дневник.
- Нажмите `+`, чтобы создать запись.
- Нажмите на запись в списке, чтобы открыть и отредактировать её.
- Нажмите на большое фото в карточке, чтобы заменить его.

## Данные

Записи хранятся только в браузере и только на устройстве, где они были созданы. Очистка данных сайта или браузера удалит записи. Большие фотографии могут не сохраниться из-за ограничения объёма `localStorage`.
